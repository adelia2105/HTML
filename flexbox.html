kesimpulan skenario 2:Dari gambar tersebut, terdapat kode CSS yang mendefinisikan gaya untuk sebuah elemen

kesimpulan skenario 3:Penambahan properti CSS pada class .wardah_flex akan membuat elemen tersebut menjadi flex container dengan child elements yang diatur menggunakan flexbox.

kesimpulan skenario 4:Dengan menambahkan flex-direction: ;, properti tersebut akan mengatur arah tata letak flexbox elemen dengan class .wardah_flex

kesimpulan skenario 5:Dengan menambahkan flex-wrap: wrap;, properti tersebut akan mengatur item-item flex dalam class .wardah_flex

kesimpulan skenario 6:Dengan menambahkan justify-content: center;, properti tersebut akan menyusun item-item flex di tengah-tengah secara horizontal dalam container flex dengan class .wardah_flex.
